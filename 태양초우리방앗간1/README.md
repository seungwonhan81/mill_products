## 🌾 프로젝트 개요

**태양초우리 방앗간**의 30년 전통과 장인정신을 담은 프리미엄 랜딩 페이지입니다. 100% 국내산 원재료와 전통 저온 압착 기술로 만든 참기름, 들기름, 고추씨를 소개하며, 건강한 식재료를 찾는 고객들에게 신뢰와 품질을 전달합니다.

### 🎯 목표 타겟

- **건강한 식재료를 찾는 주부**: 가족의 건강을 최우선으로 생각하는 분들
- **정성 어린 명절 선물을 찾는 분**: 의미 있는 선물을 원하는 분들
- **요리에 진심인 1인 가구**: 프리미엄 재료로 요리하는 것을 즐기는 분들

### 💡 핵심 메시지

1. **100% 국내산 원재료**와 **전통 저온 압착 기술**로 만든 깊은 맛
2. **내 가족이 먹는다는 마음**으로 정직하게 운영하는 방앗간의 신뢰도
3. **30년 장인 정신**을 시각적으로 구현한 브랜드 아이덴티티

## 📱 페이지 구조

### 1. **Hero Section** - 첫인상
- 슬로건: "고소함의 깊이가 다른 30년 전통 태양초우리 방앗간"
- 30년 전통 뱃지로 신뢰도 강조
- 주요 특징 아이콘 (100% 국내산, 전통 저온 압착, 정직한 제조)
- 명확한 CTA 버튼 (제품 보러가기, 전화 상담하기)
- 스크롤 인디케이터로 다음 섹션 유도

### 2. **Products Section** - 제품 소개
#### 참기름
- **황금빛 색상 강조**
- 100% 국내산 참깨 사용
- 저온 로스팅으로 고소한 향 극대화

#### 들기름
- **오메가3 풍부** 건강 기름
- 국내산 들깨 100%
- 구수한 맛과 건강 효능

#### 고추씨
- **매콤하고 깔끔한** 육수/양념용
- 국내산 고추 사용
- 깨끗한 선별 과정

### 3. **Process Section** - 제조 과정
- **4단계 제조 과정** 타임라인:
  1. 원재료 엄선
  2. 전통 저온 로스팅
  3. 저온 압착
  4. 위생적인 포장
- 30년 전통과 정직함 강조 섹션

### 4. **Testimonials Section** - 고객 후기
- 실제 고객들의 생생한 경험담
- 5점 만점 별점 표시
- 지역 정보로 신뢰도 강화

### 5. **CTA Section** - 행동 유도
- "지금 바로 주문하세요" 강력한 메시지
- 블로그 방문하기 버튼
- 전화 주문하기 버튼
- 영업시간 및 배송 정보

### 6. **Footer** - 푸터
- 브랜드 정보
- 네이버 블로그 링크
- 저작권 정보

## 📖 사용 방법

### 1. 로컬 개발 환경

프로젝트 폴더에서 `index.html` 파일을 브라우저로 열기만 하면 됩니다.

```bash
# 간단한 HTTP 서버 실행 (선택사항)
# Python 3가 설치되어 있다면:
python -m http.server 8000

# Node.js가 설치되어 있다면:
npx http-server
```

브라우저에서 `http://localhost:8000` 접속

### 2. 배포

**Publish 탭**으로 이동하여 **원클릭 배포** 가능합니다.
